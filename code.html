►How can we prepare your kid for the IT world?


The student will get to know coding while playing the role of a Knight, who is on an adventure, trying to restore an imperiled Fairy Kingdom and save the Princess.

► The course is divided into three parts

Modules 1-2 - the Coding Knight application
Children explore the concept of algorithms, memory blocks, learn loop algorithms and programs
Modules 3-6 - learning the ScratchJr commands
Use the motions and looks blocks, events, message commands, and interactivity.They can present themselves, share informations about the projects
Modules 7-8 Game mechanics and individual projects
Students implement all the skills they’ve learned and continue to apply game mechanics to create various projects
